# Notes

## DMS Default Configs for Other Apps (https://github.com/AvengeMedia/DankMaterialShell/tree/v1.4.6/core/internal/config/embedded)

## DMS Declaratively Enable Plugins

`~/.config/DankMaterialShell/plugin_settings.json`
```
{
    "pluginID": {
        "enabled": true
    }
}
```
> Get `pluginID` from `plugin.json`.

## Ghostty Run With Command
```
ghostty +new-window --wait-after-command=true -e "your-command"
```

## Install Flatpak for Current User and Auto-Yes
```
flatpak install --user -y --noninteractive flathub <app-id>
```

## `.desktop` file to run Bash script silently
```
[Desktop Entry]
Version=1.0
Type=Application
Name=My Script Launcher
Comment=Run my bash script silently
Exec=/path/to/your/script.sh
Icon=utilities-terminal
Terminal=false
Categories=Utility;
```

## Flatpak Install to User, No Interaction
```
flatpak --user install --noninteractive --assumeyes flathub org.gimp.GIMP
```
